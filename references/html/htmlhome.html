Getting Started with HTML Pages

The best place for me to start a new html page is with a blank slate. It's easy to copy and paste code you need or have already, but a good foundation is key.
If you haven't had anything generate a webpage auto-magically for you, reference this template for the basics.
It includes proper labels for most of the important elements, but you can delete them if you don't need them.

Included for quick reference at the bottom of the page is a table full of all the elements and tags we covered.

Basic Building Blocks of every HTML document

DocType Declaration

more

This is required. It must go at the top for a webpage to render properly.
<!DOCTYPE> is the declaration. As of HTML5, all that is needed is an html argument.
It is no longer necessary to specify the reference to the current version.

Your declaration should just look like this: <!DOCTYPE html>

The <html> tag

more

This is also required. It must go below the declaration, and encloses all html elements for a webpage to render properly.
<html> is the tag. It can take any global attributes.
Important attributes include hidden, lang, and style. This tag requires a closing tag.

Your code should just look like this:

<!DOCTYPE html>
<html>
 Your code goes here.
</html>

The <head> and <title> tags

more

These are also required. It must go below the declaration, and inside the <html> tags. It should enclose any metadata for search engines and web crawlers to help understand what a human is seeing.

<head> is our first tag. It can take any global attributes. None of the content in this element is visible on the actual webpage.
The tags this section expect to see are: <title>,<style>,<base>,<link>,<meta>,<script>,<noscript> can be used inside this element.

Next up is the <title> tag. It must go inside the <head> tags. It must be text only, as this is not formatted. This is what appears in the browser's tab when the webpage is open.
Search engines also use this as a title for your page. Search engines only display 50-60 charcters of the title element. There may only be one title element. It is not visible on the webpage other than the tab title.

Your code should just look like this:

<!DOCTYPE html>
<html>
 <head>
  <title>Webpage Title Goes Here</title>
 </head>
 Your code goes here.
</html>

The <body> tag

more

These is the last required tag. It must go below the declaration of the <head> tags. It should enclose any content you will display on the webpage. This is the website.
<body> is the tag. It can take any global and event attributes. All of this content is visible on the actual webpage.

Your code should just look like this:

<!DOCTYPE html>
<html>
 <head>
  <title>Webpage Title Goes Here</title>
 </head>

 <body>
  Your code goes here.
 </body>
</html>
